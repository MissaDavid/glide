# Task Completion Checklist

After creating/modifying files:
1. **Lint check**: `npm run lint` should pass with no errors
2. **Build check**: `npm run build` should succeed
3. **Type safety**: TypeScript must compile without errors
4. **Git commit**: Create atomic commits with conventional commit messages
   - Format: `<type>: <subject>` (e.g., `feat: add design tokens`)
   - Include description if needed
5. **Verification**: Self-review exported types and interfaces for correctness
