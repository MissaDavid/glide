# Suggested Commands

## Development
- `npm run dev` - Start Vite dev server with HMR
- `npm run build` - TypeScript type checking + Vite build
- `npm run lint` - Run ESLint

## Testing
- `npm run test` - Run Vitest (if configured)
- `npm run test:coverage` - Coverage reports

## Verification
After task completion:
1. Run `npm run lint` to check for style/type issues
2. Check that `npm run build` succeeds
3. If available, run tests to verify functionality

## System
- Darwin/macOS system
- Git available for commits
