# Glide Project Overview

**Purpose:** A React + TypeScript breathing exercise application (fresh scaffolding, Task 2 onwards)

**Tech Stack:**
- React 19.2.4
- TypeScript 6.0.2
- Vite 8.0.4
- Vitest + Testing Library for testing
- ESLint for linting

**Key Structure:**
- `/src` - source code (App.tsx, main.tsx, styles, assets)
- `/public` - static assets
- `package.json` - dependencies and scripts
- `tsconfig.app.json` - TypeScript config for app code
- `tsconfig.node.json` - TypeScript config for node tooling
- `vite.config.ts` - Vite configuration with vitest setup
- `eslint.config.js` - ESLint rules

**Code Style & Conventions:**
- TypeScript strict mode
- React functional components with hooks
- ESLint configured with React rules
- CSS modules or CSS files alongside components
